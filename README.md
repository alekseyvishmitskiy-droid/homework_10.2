Модуль маски: Реализовать в этом модуле две функции: 1) Функцию маскировки номера банковской карты get_mask_card_number. Функция принимает входной номер карты и возвращает ее маску. Номер карты замаскирован и отображается в формате XXXX XX** **** XXXX, где X — это цифра номера. В результате есть первые 6 цифр и последние 4 цифры, остальные символы указаны звездочками, номер разбития по блокам по 4 цифрам, разделенным пробелами. 2)Функция маскировки номера банковского счета get_mask_account. Функция принимает на вход номер счета и возвращает ему маску. Номер счета замаскирован и отображается в формате **XXXX, где X — это цифра номера. То есть есть только последние 4 цифры номера, а перед ними — две звездочки.

Модуль виджета: функция Mask_account_card принимает один аргумент — символ, содержащий тип и номер карты или счета. Аргументом может быть строка типа Visa Platinum 7000792289606361, или Maestro 7000792289606361, или Счет 73654108430135874305. Разделять пример на 2 аргумента (отдельно). имя, отдельный номер) нельзя! Функция маска_аккаунт_карта принимает на вход код с датой в формате «2024-03-11T02:26:18.671407» и принимает код с датой в формате «ДД.ММ.ГГГГ» («11.03.2024»).

Модуль обработки: функция filter_by_state принимает список слов и опциональное значение для включения состояния (по умолчанию «EXECUTED»). Функция возвращает новый список слов, включающий только те словари, в которых состояние ключа соответствует указанному результату.

Функция sort_by_date учитывает список слов и необязательный параметр, задающий порядок сортировки (по умолчанию — убывание). функция возвращает новый список, отсортированный по дате (дате).
